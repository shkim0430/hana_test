# RepayFirst AI — MVP Landing Page

Single-file fintech B2B landing page for market validation + lead generation.

## Run locally
- Open `index.html` in any browser (no install, no build).

## Deploy with Vercel (GitHub)
1. Push this folder to a GitHub repo
2. In Vercel: **New Project** → Import the repo
3. Framework preset: **Other**
4. Build command: **None**
5. Output directory: **.** (repo root)

This is a static site (HTML/CSS/vanilla JS only).
